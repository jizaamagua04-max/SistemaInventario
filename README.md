# Sistema de Inventario

Sistema web de gestión de inventario desarrollado con PHP y MySQL para el control de productos, ventas, usuarios y reportes.

## Funcionalidades

* Gestión de productos
* Gestión de categorías
* Gestión de usuarios
* Registro de ventas
* Reportes diarios
* Reportes mensuales
* Control de inventario

## Tecnologías utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

## Objetivo

Desarrollar una solución para la administración eficiente de inventarios, permitiendo controlar productos, ventas y usuarios desde una interfaz web intuitiva.
